# FuCrawler

基于 nodejs 的分布式爬虫，封装常见金融行情业务

养一只 nodejs 爬虫溜达中国证券网，再造一个'东方财富'也不在话下

## 主要业务

- 行业新闻资讯爬取

- 指数（道琼斯）（恒生指数）（上证50）

- 走势图表
